## 🚀 The 8 Steps You Need to Do

1. **Install Java 17** (for backend)
2. **Install Node.js** (for frontend)
3. **Install Maven** (to build Java project)
4. **Set up MongoDB Atlas** (free cloud database)
5. **Configure the project** (add MongoDB connection)
6. **Install frontend dependencies** (`npm install`)
7. **Start backend** (`mvn spring-boot:run`)
8. **Start frontend** (`npm start`)
